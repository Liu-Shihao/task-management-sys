# 技术规格说明书 (Spec)

## 1. 项目概述

**项目名称**: FinBlock 任务管理系统  
**项目类型**: Web 管理系统 (B端企业应用)  
**核心功能**: 支持 Excel 导入、模板管理、发布 rundown 生成、自动化任务执行 (Jenkins/Ansible)  
**目标用户**: FinBlock 运维/部署工程师

---

## 2. 需求对齐分析

### 2.1 核心功能模块

| 用户故事 | 功能点 | 优先级 |
|---------|--------|--------|
| US1 | Excel 文件上传 (.xlsx, .xls) | P0 |
| US2 | 模板管理 (创建、克隆) | P0 |
| US3 | 从模板生成 Release Rundown | P0 |
| US4 | 任务控制 (Run/Edit/Delete) | P0 |
| US5 | 自动化执行 (Jenkins/Ansible) | P0 |

### 2.2 非功能性需求

| 需求类型 | 描述 |
|---------|------|
| **可用性** | UI 响应时间 < 200ms |
| **可靠性** | 任务状态实时更新 (Near Real-time) |
| **扩展性** | 支持插件化扩展其他 CI/CD 工具 |
| **数据一致性** | 任务状态与外部系统 (Jenkins/Ansible) 最终一致 |
| **安全性** | 文件上传校验、权限控制 |

### 2.3 用户角色

| 角色 | 权限 |
|------|------|
| Viewer | 查看 rundown、模板 |
| Operator | 执行任务、编辑任务 |
| Admin | 模板管理、上传 Excel、全量权限 |

---

## 3. 业务流程

```
[Excel导入] → [模板管理] → [生成Rundown] → [任务控制] → [自动化执行]
     ↑                                      ↓
     └──────────── [执行结果回写] ←─────────┘
```

---

## 4. 验收标准

- [ ] 支持 .xlsx/.xls 文件上传，错误格式提示明确
- [ ] 模板支持创建、克隆、编辑、删除
- [ ] 生成的 Rundown 包含唯一标识，状态为 "Pending"
- [ ] 任务支持 Run/Edit/Delete 操作
- [ ] Jenkins/Ansible 执行结果实时回显
- [ ] 失败任务可查看错误日志
